TURINIO APRAŠYMAS


	Where Would You Travel yra tematinis paveiksleliu talpinimo puslapis, skirtas talpinti paveikslelius susijusius su kelionemis, idomiomis bei lankytinomis vietomis.

	Browse mygtuko pagalba vartotojas gali pasirinkti paveiksleli ikelimui i puslapi.

	Teksto skiltyje vartotojas gali aprašyti savo ispudžius susijusius su kelione.

	Upload mygtuko pagalba vartotojas gali patalpinti savo paveiksleli bei parašyta teksta puslapyje. Turinys taps prieinamas visiems puslapio vartotojams.

	Puslapio apacioje matomas puslapiavimas, kurio pagalba galima pasiekti senesnius paveikslelius. Naujausias paveikslelis visuomet matomas pirmo puslapio viršuje.


PROJEKTO APRAŠYMAS

	Projektas parašytas naudojant HTML5, CSS3 ir PHP7.

	Kodas patalpintas GitHub prieeigoje: https://github.com/lnsyesterday/img_uploader.

	Lokaliai projekta galima pasileisti naudojant Xampp arba kitus panašaus pobudžio lokalius serverius.
